Definition : Chordless Cycle

A chordless cycle of a graph GGG is a graph cycle of length at least four in GGG that has no cycle chord (i.e., the graph cycle is an induced subgraph).

Definition : Chordal Graph

A chordal graph is a simple graph possessing no chordless cycles. A chordless cycle is sometimes also called a graph hole (Chvátal).

Definition : Cycle Chord

A chord of a graph cycle CCC is an edge not in the edge set of CCC whose endpoints lie in the vertex set CCC.
