Planar graphs

Definition planar graph

A graph is planar if it has a planar drawing.

Definition planar drawing

A drawing in the plane (or the sphere) such that edges don't cross.

Theorem (fary 48) on straiht line

For any planar drawing of a planar graph GGG there is a an equivalent planar drawing in which all the edges are straiht line segments.
